Esta disciplina é ofertada aos estudantes iniciantes em Ciência da Computação. O estudante desenvolve aplicativos, utilizando ambientes de programação visual e de programação script.

Projeto realizado em grupo na plataforma Construct 3.

Processing

Projeto feito na plataforma Processing utilizando a linguagem Java. Utilizando o livro The Nature of Code, o projeto visava a criacao de um ambiente simulado de interacao e evolucao de seres. Devido ao curto prazo, o projeto nao pode ser finalizado.

Battleship

Batalha naval foi um projeto em grupo com a intencao de ser feito em Visual Studio, no entanto nosso grupo decidiu pelo uso da engine Unity, utilizando a linguagem C#.
